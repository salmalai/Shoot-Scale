# Format AI Prompt — exact output template

Produce the format using EXACTLY this template and nothing else. The `---` fences are part
of the output.

```
---
You write ONE short-form video script in a single fixed format: the "[name the format in 2–4 words]".

HOW THE FORMAT WORKS (picture it):
[3–5 sentences. The structure, the on-screen visual, how many beats, how it escalates/turns, and why it grabs attention. Written so an AI that has never seen this format fully understands how to execute it.]

CHOOSING THE TOPIC (keep it niche-agnostic):
[2–4 lines encoding the topic rules. State that this engine works in ANY niche and only the topic changes. Demand a massive-TAM premise that grabs someone with zero interest in the niche; a curiosity gap (you can't guess the content from the title); and a relatable/self-implicating angle that makes the viewer wonder if it applies to them. Name the universal levers to pull: money, status, freedom, health, secrets, pop-culture, "wait, that's true?", everyday behavior.]

STRUCTURE / MAD LIB:
[the fill-in-the-blank skeleton with [bracketed] variables and the correct number of beats]

REFERENCE TRANSCRIPT — match its rhythm and length, NOT its words:
[the clean original transcript]
---
```

Rules:
- Copy-paste ready, under ~250 words.
- Short, memorable format name.
- The CHOOSING THE TOPIC block is required on every brick — it carries the format's own
  optimization guidance (TAM, curiosity gap, relatability).
- No "Your Job," task, or output section — end at the reference transcript.
- Nothing outside the template.

## Filled example (for reference — do not copy its words, match its shape)

It grabs attention because viewers see something they recognize fast, and the snap judgments invite agreement or outrage. Escalate by opening on the most famous item at top tier, dropping 1–2 contrarian low-tier "must-reads" to spark debate, then building to a beat tied to a huge personal payoff before a respectful low-tier closer.

This engine works in any niche — only the list changes (books, tools, habits, mistakes, purchases). Pick a list category everyone already has opinions on, so a total outsider still wants to watch. Lean on money, status, or time saved.
